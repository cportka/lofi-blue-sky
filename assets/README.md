# assets

Two clear buckets, plus the palette sheet:

| folder | what's in it |
|--------|--------------|
| **`skies/`** | the **historical released editions** — the *lofi blue sky #N* sky loops (gifs) from the collections. Inspiration + museum source media. See [`skies/README.md`](./skies/README.md). |
| **`renders/`** | stills the **generator** produced — the featured `hero` / `filmstrip` / `billow`, and the per-version design-log stills (`v0.1-…`, `v0.3-…`). |
| **`palettes/`** | the palette reference sheet (`palettes.svg` / `.json`), generated from the core by `scripts/palette-swatches.mjs`. |
