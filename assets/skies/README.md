# skies — the historical released editions

The source aesthetic: **released *lofi blue sky #N* editions** — 400×400 seamless sky loops, a decade
of filmed sky processed into the project's signature glitch (datamosh, pixel-sort, slit-scan,
downsample). These are the loops the generator distills into math ([../../docs/AESTHETIC.md](../../docs/AESTHETIC.md)),
and the media the [museum](../../docs/MUSEUM.md) is built around.

All are 400×400. Present here (a subset of the collections): editions
**1–13, 16–22, 28, 30, 31** (23 loops). Notably, **#19** carries a grid annotation in its source
name — **`1x9`** — a direct nod to the pixel-split idea now generalized in the Genesis engine
(varying the horizontal × vertical split).

> **Size note:** these are ~34 MB of gifs (largest ~12 MB). They're committed as the project's
> reference library; if the set grows large, move it to Git LFS or link out to the collections
> instead of vendoring the media.

The three loops the build brief singles out as the engines' direct ancestors:

- **`32__OG`** — horizontal slit-scan sunset → **Genesis**.
- **`35`** — datamosh blue sky + salmon clouds → **Billow**.
- **`31`** — olive/mauve downsample mosaic → the experimental **mosaic** mode.
